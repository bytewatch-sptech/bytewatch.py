# bytewatch.py
Repositório do projeto de monitoramento de servidores do Tiktok usando o Python

## Finalidades
- Capturar informações dos componentes de hardware usando o psutil
- Salvar as informações em um arquivo csv
- Realizar a leitura do arquivo csv e formatar os dados
- Conversar com o Banco de dados para colher informações sobre o servidor
- Salvar arquivos csv no serviço bucket da AWS
